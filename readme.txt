With RoundIt plugin you can set special effects to your pictures that you add in pages or posts. You can make your images Round, set Round Corners, add border with color and add a shadow to your image.

== Description ==

No more photshop skills required to give your images a fancy look. With RoundIt you can make your images Round, set Round Corners, add border with color and add a shadow to your image. The plugin adds these features in the media manager where you upload the image files. You can set different roundness, border or shadow effect settings for different images by entering the desired input in the new fields. 

**Features**<br>
1. Make Images round or set round corners<br>
2. Add borders alongwith border color and width<br>
3. Add shadow   <br>
4. Set different effects for different pictures  <br>

== Installation ==

1.Login to your wordpress website<br><br>
2.Goto Plugins >> Add New<br><br>
3.Click Upload  <br><br>
4.Browse and Locate the roundit.zip file on your harddisk and click Install Now button.<br> <br>
